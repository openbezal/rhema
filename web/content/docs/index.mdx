Rhema listens to a live sermon audio feed, transcribes speech in real time,
detects Bible verse references — both explicit citations and quoted
passages — and renders them as broadcast-ready overlays via NDI for live
production.

![Rhema desktop app — live transcript on the left, detections in the centre, queue and live preview on the right](/docs/screenshots/dashboard-overview.png)

## Get started in two commands

<Callout title="Prerequisites" type="info">
You'll need [Bun](https://bun.sh), the [Rust toolchain](https://rustup.rs),
[Python 3](https://www.python.org/), and your platform's
[Tauri v2 prerequisites](https://v2.tauri.app/start/prerequisites/) installed
first. See [Prerequisites](/docs/getting-started/prerequisites) for the full
list.
</Callout>

```bash
git clone https://github.com/openbezal/rhema.git && cd rhema
bun install && bun run setup:all
bun run tauri dev
```

That's it. `setup:all` runs seven idempotent phases — Python venv, Bible
downloads, SQLite build, ONNX export, embeddings — and `tauri dev` boots the
desktop app. See the [installation guide](/docs/getting-started/installation)
for a guided walkthrough.
